## 1 问题描述

HR_S50-2000 码垛工作站由四个组件构成：机器人底座电箱（固定）、码垛蓝色框架、传送带和托盘。目标是 **优化框架和传送带的位置**，使得：

1. 机械臂可从传送带拾取箱子并放置到框架内 **任意位置和任意层数**（2×2×3 = 12 箱/周期）；
2. **优先最大化安全与可靠性**（框架/传送带离电箱更远，拾取与入口避障净裕量更大）；
3. 所有放置位存在 **无碰撞可行路径**（可达、避障、入口走廊充裕）；
4. **运输路径优先采用"收缩通过"策略** —— 机械臂收缩手臂从框架前方开口进入，而非手臂张开从框架顶部翻越。

**不变量**：
- 电箱位于坐标原点，尺寸 550×650×800 mm（长条状），不可移动
- 框架尺寸 1200×650×2000 mm 固定，无中部横梁
- 传送带尺寸 550×3500 mm 固定
- 箱子尺寸 350×280×250 mm 固定
- 托盘高度固定（世界坐标 1000 mm，基座坐标 $h_p$ = 200 mm）

## 5 衍生量

### 5.1 框架立柱 XY 坐标

$$\text{col}_{1,2} = \left(\mp\tfrac{W_f}{2},\; y_f - \tfrac{D_f}{2}\right), \qquad
  \text{col}_{3,4} = \left(\mp\tfrac{W_f}{2},\; y_f + \tfrac{D_f}{2}\right)$$

立柱 Z 范围: $[-800,\; 1200]$ mm。

### 5.2 箱子放置位 TCP 网格

$$x_b = \left(i_x - \frac{n_x+1}{2}\right)(l_x + g_{box}), \qquad
  y_b = y_f + \left(i_y - \frac{n_y+1}{2}\right)(w_y + g_{box}), \qquad
  z_b = h_p + \left(i_z - \frac{1}{2}\right) h_z$$

$x_b \in \{-180, +180\}$ mm, $y_b \in \{y_f - 145, y_f + 145\}$ mm。  
$z_b \in \{325, 575, 825\}$ mm（L1/L2/L3 层心）。

### 5.3 拾取位 TCP

$$\mathbf{p}_{pick} = \left(x_c,\; y_c,\; z_{conv} + \frac{h_z}{2}\right) = (x_c,\; y_c,\; 140) \text{ mm}$$

### 5.4 框架入口航路点 (v3.0 新增)

每个放置位 $b$ 对应一个航路点位于框架前入口处：

$$\mathbf{w}_b = \left(x_b,\; y_f - \frac{D_f}{2},\; z_b + \Delta z_{app}\right)$$

航路点 X 与放置位对齐，Y 在框架前道（两近端立柱连线处），Z 高于放置位 $\Delta z_{app}$。

> **物理含义**: 航路点模拟"收缩通过"策略中机械臂经由框架前开口进入的必经点。机械臂在此处收缩手臂，通过两根近端立柱之间的 $1100$ mm 宽通道进入框架内部。

### 5.5 工具碰撞球中心

放置第 $b$ 个箱子时：$\mathbf{t}_b = \mathbf{p}_b + (0, 0, -\delta_{tool})$

航路点处：$\mathbf{t}_b^{wp} = \mathbf{w}_b + (0, 0, -\delta_{tool})$

## 6 目标函数

v3.1 将优化目标改为“**避碰优先，路径次优先**”，与实际工程要求一致：先保证可靠可行，再优化效率。

### 6.1 Safety-first 主目标 (v3.1 核心)

定义安全评分（越大越安全）：

$$S(\mathbf{x}) = w_1 \cdot \tilde g_{fc} + w_2 \cdot \tilde g_{cc} + w_3 \cdot \tilde m_{pick} + w_4 \cdot \tilde m_{wp}$$

其中：

- $g_{fc}$：框架-电箱净间距
- $g_{cc}$：传送带-电箱净间距
- $m_{pick}$：拾取位工具球到最近立柱净裕量
- $m_{wp}$：框架入口航路点工具球到最近立柱净裕量

为避免“无限拉远”造成无意义扩张，采用饱和形式：

$$\tilde g_{fc}=\min(g_{fc}, g_{fc}^{tar}),\;\tilde g_{cc}=\min(g_{cc}, g_{cc}^{tar}),\;\tilde m_{pick}=\min(m_{pick}, m_{pick}^{tar}),\;\tilde m_{wp}=\min(m_{wp}, m_{wp}^{tar})$$

最终最小化目标：

$$\min_{\mathbf{x}} J(\mathbf{x}) = -S(\mathbf{x}) + \lambda \cdot \frac{L_{wp}(\mathbf{x})}{1000}$$

其中 $L_{wp}$ 为航路点循环路径长度（次级项），$\lambda$ 为小权重。

### 6.2 航路点路径代理 (次级项)

真实码垛路径经由框架前开口进入/退出（"收缩通过"策略），而非直线飞越。v3.0 引入框架入口航路点，使路径代理更接近实际：

$$\min_{\mathbf{x}} \; f(\mathbf{x}) = \sum_{b=1}^{N} \left[ d_{HP} + d_{PW}^{(b)} + 2 \cdot d_{WB}^{(b)} + d_{WH}^{(b)} \right] + f_{penalty}$$

其中各段距离：

| 段 | 符号 | 路径 | 含义 |
|----|------|------|------|
| HOME→Pick | $d_{HP}$ | $\|\mathbf{p}_{HOME} - \mathbf{p}_{pick}\|$ | 移向传送带 (所有箱相同) |
| Pick→Waypoint | $d_{PW}^{(b)}$ | $\|\mathbf{p}_{pick} - \mathbf{w}_b\|$ | 运输到框架前门 |
| Waypoint→Place | $d_{WB}^{(b)}$ | $\|\mathbf{w}_b - \mathbf{p}_b\|$ | 进入框架到位 |
| Place→Waypoint | $d_{WB}^{(b)}$ | = 同上 (对称) | 退出到框架前门 |
| Waypoint→HOME | $d_{WH}^{(b)}$ | $\|\mathbf{w}_b - \mathbf{p}_{HOME}\|$ | 返回初始位 |

**每箱 5 段，其中入框和出框用同一航路点（对称）故内段计 2 倍。**

### 6.3 收缩通过 vs 顶部翻越分析

| 策略 | 路径特征 | 最大 Z | 路径长度 | 风险 |
|------|---------|--------|---------|------|
| **收缩通过** (优选) | 经框架前开口, 臂水平 | $z_b + \Delta z_{app} \leq 975$ | **短** | 低 |
| 顶部翻越 (避免) | 翻越框架顶梁 $Z=1200$ | $\geq 1200 + r_{tool} = 1425$ | 长 +1000mm | 高 |

航路点目标函数自动偏好"收缩通过"：因为航路点 $z_{wp} \leq 975$ mm 远低于 $Z_{top}=1200$ mm，优化结果天然避免需要顶部翻越的布局。

### 6.4 软惩罚项

$$f_{penalty} = w_p \cdot \max\left(0,\; c_{threshold} - c_{\min}\right)$$

$c_{\min}$ = 所有放置位工具球到最近柱的最小 XY 净间隙。

## 7 约束条件

所有约束以 $c_i(\mathbf{x}) \leq 0$ 形式表达。

### C1 — 框架-电箱 Y 分离

$$\frac{D_{cab}}{2} + g_{\min} - \left(y_f - \frac{D_f}{2}\right) \leq 0 \qquad \Rightarrow \quad y_f \geq 680$$

### C2 — 传送带-电箱 X 分离

$$\frac{W_{cab}}{2} + g_{\min} - \left(x_c - \frac{W_{conv}}{2}\right) \leq 0 \qquad \Rightarrow \quad x_c \geq 580$$

### C3 — 框架立柱-传送带分离 (4 约束)

对每根柱 $k$, 计算柱中心到传送带矩形的穿透深度 $d_k$:()

- $d_k > 0$ (柱在传送带内): $g_{\min} - d_k \leq 0$
- $d_k \leq 0$ (柱在传送带外): 自动满足

### C4 — 放置位可达性 (48 约束)

对每个放置位 $b$:

$$\|\mathbf{p}_b^{xy}\| \leq \eta_{place} \cdot R_{\max} = 1560 \tag{C4a}$$
$$\|\mathbf{p}_b^{xy}\| \geq R_{\min} = 200 \tag{C4b}$$
$$z_b \leq Z_{top} - r_{col} - \epsilon_{top} = 1120 \tag{C4c}$$
$$z_b \geq -600 \tag{C4d}$$

> 代入 $h_p=200$: 最高 $z_b = 825$ mm ≤ 1120 ✓ 始终满足。C4c 对 $y_f$ 无约束。

### C5 — 拾取位可达性 (2 约束)

$$\|\mathbf{p}_{pick}^{xy}\| \leq \eta_{pick} \cdot R_{\max} = 1400 \tag{C5a}$$
$$\|\mathbf{p}_{pick}^{xy}\| \geq R_{\min} = 200 \tag{C5b}$$

### C6 — 工具球-立柱碰撞间隙 (48 约束)

$$r_{tool} + r_{col} + \epsilon_{col} - \|\mathbf{t}_b^{xy} - \text{col}_k^{xy}\| \leq 0, \quad \forall b, k$$

所需最小距离: $225 + 50 + 50 = 325$ mm。

### C7 — 顶梁间隙 (含接近点, HP 固定)

L3 接近点工具球最高点:

$$z_{tool,top}^{app} = h_p + (n_z - 0.5) h_z + \Delta z_{app} - \delta_{tool} + r_{tool}
= 200 + 625 + 150 - 125 + 225 = 1075 \text{ mm}$$

$$z_{tool,top}^{app} + \epsilon_{top} \leq Z_{top} - r_{col}$$
$$1075 + 30 = 1105 \leq 1150 \quad \checkmark \quad (\text{余量 45 mm})$$

> **v3.0 变更**: $\Delta z_{app}$ 从 200 → 150 mm, $\epsilon_{top}$ 从 50 → 30 mm。  
> 原 v2.0 中 $h_p$ 可优化, v3.0 固定 $h_p=200$ 后需此调整。  
> 物理合理性: 150mm 接近高度足以容纳箱间间隙和吸放动作, 30mm 顶梁余量配合 TCP 偏航可提供足够安全裕度。

C7 不含决策变量（$h_p$ 固定），为常量检查。

### C8 — 入口走廊宽度 (常量)

$$2 r_{tool} + \epsilon_{entry} = 500 \leq W_f - 2 r_{col} = 1100 \quad \checkmark$$

### C9 — 入口走廊高度

$$2 r_{tool} + \epsilon_{top} \leq (Z_{top} - r_{col}) - h_p$$
$$480 \leq 1150 - 200 = 950 \quad \checkmark$$

C9 不含活跃决策变量（$h_p$ 固定），为常量检查。

### C10 — 箱子在框架内 (常量)

X 跨 = 355 ≤ 570 ✓, Y 跨 = 285 ≤ 295 ✓

### C11 — 拾取位在框架外部

$$\max\!\left(x_c - \tfrac{W_f}{2} - m_{pick},\; \left(y_f - \tfrac{D_f}{2}\right) - m_{pick} - y_c,\; y_c - \left(y_f + \tfrac{D_f}{2}\right) - m_{pick}\right) \geq 0$$

### C12 — 拾取位工具球-立柱碰撞间隙 (4 约束)

$$r_{tool} + r_{col} + \epsilon_{col} - \|\mathbf{t}_{pick}^{xy} - \text{col}_k^{xy}\| \leq 0, \quad \forall k$$

### C13 — 航路点可达性 (v3.0 新增, 24 约束)

框架入口航路点 $\mathbf{w}_b$ 必须在机械臂工作范围内:

$$\|\mathbf{w}_b^{xy}\| \leq \eta_{wp} \cdot R_{\max} = 1200 \tag{C13a}$$
$$\|\mathbf{w}_b^{xy}\| \geq R_{\min} = 200 \tag{C13b}$$

代入 $\mathbf{w}_b = (x_b, y_f - D_f/2, \cdot)$:

$$\sqrt{x_b^2 + (y_f - D_f/2)^2} \leq 1200$$

> **物理含义**: 航路点在框架入口处，比放置位更靠近机器人。使用较低的臂展利用率 $\eta_{wp}=0.60$ 确保在此处手臂有足够收缩裕度（不需满伸即可到达）。

### C14 — 运输高度限制 (v3.0 新增, 12 约束)

航路点 Z 高度不得超过顶梁内侧，确保"收缩通过"路径可行:

$$z_{wp}^{(b)} = z_b + \Delta z_{app} \leq Z_{top} - r_{col} - \epsilon_{top} = 1120 \tag{C14}$$

代入: 最大 $z_{wp} = 825 + 150 = 975$ mm ≤ 1120 ✓

> **v3.0 说明**: 此约束确保即使最高层 (L3) 的航路点也在顶梁下方, 机械臂无需翻越框架顶部。C14 不含活跃决策变量（$h_p$ 固定），为常量验证。

### C15 — 框架入口通道净宽 (v3.0 新增, 12 约束)

航路点处的工具碰撞球必须能通过两近端立柱之间:

$$r_{tool} + r_{col} + m_{entry} - \|\mathbf{t}_{wp,b}^{xy} - \text{col}_{1,2}^{xy}\| \leq 0$$

工具球心在航路点: $\mathbf{t}_{wp,b} = (x_b, y_f - D_f/2, z_{wp} - \delta_{tool})$

需要检查 vs col1 和 col2 (近端两柱, 同 Y):

$$\text{For } k \in \{1, 2\}: \quad \sqrt{(x_b - \text{col}_k^x)^2 + 0^2} \geq r_{tool} + r_{col} + m_{entry} = 325$$

代入: 柱 1 在 $X = -600$, 柱 2 在 $X = +600$。$x_b \in \{-180, +180\}$:
- $|{-180} - ({-600})| = 420 \geq 325$ ✓
- $|{+180} - ({+600})| = 420 \geq 325$ ✓

> C15 不含活跃决策变量（$x_b$ 由固定尺寸决定），为常量验证。但代码中仍保留为安全检查。

## 附录 A — 推导细节

### A.1 航路点路径 vs 直线路径长度

**直线路径** (v2.0): $f_b = d_{HP} + d_{PB} + d_{BH}$

**航路点路径** (v3.0): $f_b = d_{HP} + d_{PW} + 2 d_{WB} + d_{WH}$

航路点路径一般长于直线路径（加入了经由框架入口的迂回），但更准确地反映了实际"收缩通过"路径长度。优化结果因此偏向缩短 Pick → 框架入口 和 框架入口 → Place 的距离，而非不切实际的 Pick → Place 直线。

### A.2 工具球最高点验证 ($h_p=200$)

$$z_{tool,top}^{app} = 200 + 625 + 150 - 125 + 225 = 1075 \text{ mm}$$
$$Z_{beam,inner} = 1200 - 50 = 1150 \text{ mm}$$
$$\text{净间隙} = 1150 - 1075 = 75 \text{ mm} > \epsilon_{top} = 30 \text{ mm} \quad \checkmark$$
