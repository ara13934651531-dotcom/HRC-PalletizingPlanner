## 1 问题描述

HR_S50-2000 机器人码垛工作站由四个可调组件构成：机器人底座电箱（固定）、码垛蓝色框架、传送带和托盘。目标是**优化框架、传送带和托盘的位置**，使得：

1. 机械臂可从传送带拾取箱子并放置到框架内**任意位置和任意层数**（2×2×3 = 12 箱/周期）；  
2. 所有 12 个拾取—放置循环的**路径总长度最小化**；  
3. 所有放置位存在**无碰撞可行路径**（可达、避障、入口走廊充裕）。

**约束**：电箱（机器人基座）位于坐标原点，不可移动。框架无中部横梁（v3.0 确认）。

## 5 衍生量

### 5.1 框架立柱 XY 坐标

四根立柱位于框架角点：

$$\text{col}_{1} = \left(-\tfrac{W_f}{2},\; y_f - \tfrac{D_f}{2}\right), \quad
  \text{col}_{2} = \left(+\tfrac{W_f}{2},\; y_f - \tfrac{D_f}{2}\right)$$

$$\text{col}_{3} = \left(-\tfrac{W_f}{2},\; y_f + \tfrac{D_f}{2}\right), \quad
  \text{col}_{4} = \left(+\tfrac{W_f}{2},\; y_f + \tfrac{D_f}{2}\right)$$

立柱 Z 轴范围：$[Z_{bot},\; Z_{top}] = [-H_{cab},\; H_f - H_{cab}] = [-800,\; 1200]$ mm。

### 5.2 箱子放置位 TCP 网格

对于第 $(i_x, i_y, i_z)$ 个箱子（$i_x \in \{1,\dots,n_x\}$, $i_y \in \{1,\dots,n_y\}$, $i_z \in \{1,\dots,n_z\}$）：

$$\Delta_x = l_x + g_{box}, \qquad \Delta_y = w_y + g_{box}$$

$$x_b = \left(i_x - \frac{n_x+1}{2}\right) \Delta_x$$

$$y_b = y_f + \left(i_y - \frac{n_y+1}{2}\right) \Delta_y$$

$$z_b = h_p + \left(i_z - \frac{1}{2}\right) h_z$$

TCP 坐标：$\mathbf{p}_b = (x_b, y_b, z_b)$。

> 示例（$n_x=2, n_y=2$）：$x_b \in \{-180, +180\}$ mm, $y_b \in \{y_f - 145, y_f + 145\}$ mm。

### 5.3 拾取位 TCP

$$\mathbf{p}_{pick} = \left(x_c,\; y_c,\; z_{conv} + \tfrac{h_z}{2}\right) = \left(x_c,\; y_c,\; 140\right) \text{ mm}$$

### 5.4 工具碰撞球中心

放置第 $b$ 个箱子时，工具球中心：

$$\mathbf{t}_b = \mathbf{p}_b + (0, 0, -\delta_{tool})$$

放置接近点（TCP 抬高 $\Delta z_{app}$）时的工具球中心：

$$\mathbf{t}_b^{app} = \mathbf{p}_b + (0, 0, \Delta z_{app} - \delta_{tool})$$

## 6 目标函数

最小化 12 箱码垛循环的**总笛卡尔路径代理**（直线距离之和）：

$$\min_{\mathbf{x}} \; f(\mathbf{x}) = \sum_{b=1}^{N} \Big[ \underbrace{\|\mathbf{p}_{HOME} - \mathbf{p}_{pick}\|}_{d_{HP}} + \underbrace{\|\mathbf{p}_{pick} - \mathbf{p}_b\|}_{d_{Pb}} + \underbrace{\|\mathbf{p}_b - \mathbf{p}_{HOME}\|}_{d_{bH}} \Big] + f_{penalty}$$

由于 $d_{HP}$ 对所有箱子相同，可简化为：

$$f(\mathbf{x}) = N \cdot d_{HP} + \sum_{b=1}^{N} \left( d_{Pb} + d_{bH} \right) + f_{penalty}$$

### 6.1 软惩罚项

鼓励更大的最小碰撞间隙，提高后续路径规划的成功率：

$$f_{penalty} = w_p \cdot \max\left(0,\; c_{threshold} - c_{\min}\right)$$

其中：
- $c_{\min} = \min_{b,k} \left( \|\mathbf{t}_b^{xy} - \text{col}_k^{xy}\| - r_{tool} - r_{col} \right)$ — 所有放置位工具球到最近柱的最小净间隙
- $c_{threshold} = 300$ mm — 期望最小间隙阈值
- $w_p = 500$ — 惩罚权重

> **备注**：笛卡尔距离是实际关节空间路径长度的合理代理。对全局布局优化而言，此简化足以捕捉主要趋势。

## 7 约束条件

所有约束以 $c_i(\mathbf{x}) \leq 0$ 形式表达。

### C1 — 框架-电箱不重叠（Y 分离）

框架近端 Y 与电箱后端 Y 间需保持物理间距：

$$\underbrace{\frac{D_{cab}}{2}}_{325} + g_{\min} - \left(y_f - \frac{D_f}{2}\right) \leq 0$$

$$\Rightarrow \quad y_f \geq \frac{D_{cab} + D_f}{2} + g_{\min} = 650 + g_{\min}$$

### C2 — 传送带-电箱不重叠（X 分离）

$$\frac{W_{cab}}{2} + g_{\min} - \left(x_c - \frac{W_{conv}}{2}\right) \leq 0$$

$$\Rightarrow \quad x_c \geq \frac{W_{cab} + W_{conv}}{2} + g_{\min} = 550 + g_{\min}$$

### C3 — 框架立柱-传送带分离（固定 4 约束）

对每根柱 $k$，计算柱中心到传送带 XY 矩形的穿透深度 $d_k$：

$$d_k = \min\!\left(\text{col}_k^x - x_{c,L},\; x_{c,R} - \text{col}_k^x,\; \text{col}_k^y - y_{c,B},\; y_{c,T} - \text{col}_k^y\right)$$

其中 $x_{c,L/R} = x_c \mp W_{conv}/2$，$y_{c,B/T} = y_c \mp L_{conv}/2$。

- $d_k > 0$：柱在传送带矩形内，要求 $d_k \geq g_{\min}$（约束：$g_{\min} - d_k \leq 0$）
- $d_k \leq 0$：柱在传送带矩形外，约束自动满足（$d_k \leq 0$）

> **v2.0 修复**：始终输出固定 4 条约束（每柱一条），解决 v1.0 中条件判断导致的约束向量变长缺陷。

### C4 — 放置位可达性（12 个约束组）

对每个放置位 $b \in \mathcal{B}$：

$$\|\mathbf{p}_b^{xy}\| \leq \eta_{place} \cdot R_{\max} \tag{C4a}$$

$$\|\mathbf{p}_b^{xy}\| \geq R_{\min} \tag{C4b}$$

$$z_b \leq Z_{top} - r_{col} - \epsilon_{top} \tag{C4c}$$

$$z_b \geq -600 \tag{C4d}$$

### C5 — 拾取位可达性

$$\|\mathbf{p}_{pick}^{xy}\| \leq \eta_{pick} \cdot R_{\max} \tag{C5a}$$

$$\|\mathbf{p}_{pick}^{xy}\| \geq R_{\min} \tag{C5b}$$

### C6 — 工具球-立柱碰撞间隙

对每个放置位 $b$ 和每根柱 $k$（XY 平面距离，因柱为竖直线段）：

$$r_{tool} + r_{col} + \epsilon_{col} - \|\mathbf{t}_b^{xy} - \text{col}_k^{xy}\| \leq 0 \tag{C6}$$

### C7 — 顶梁间隙（含接近点）

L3 层放置接近点的工具球最高点不超过顶梁内侧：

$$z_{tool,top}^{app} = h_p + \left(n_z - \tfrac{1}{2}\right) h_z + \Delta z_{app} - \delta_{tool} + r_{tool}$$

$$z_{tool,top}^{app} + \epsilon_{top} \leq Z_{top} - r_{col} \tag{C7}$$

代入数值：

$$h_p + 925 \leq 1150 - \epsilon_{top} \quad\Rightarrow\quad h_p \leq 225 - \epsilon_{top}$$

### C8 — 入口走廊宽度

近端两柱间净宽度需容纳工具球通过：

$$2 r_{tool} + \epsilon_{entry} \leq W_f - 2 r_{col} \tag{C8}$$

代入：$2 \times 225 + 50 = 500 \leq 1200 - 100 = 1100$ ✓（不含决策变量，始终满足）

### C9 — 入口走廊高度

从托盘面到顶梁的净高度需容纳工具球通过：

$$2 r_{tool} + \epsilon_{top} \leq (Z_{top} - r_{col}) - h_p \tag{C9}$$

$$\Rightarrow \quad h_p \leq Z_{top} - r_{col} - 2 r_{tool} - \epsilon_{top} = 1150 - 450 - \epsilon_{top}$$

### C10 — 箱子在框架内

最外侧箱子边缘不超出框架内壁（扣管材半径）：

$$\frac{(n_x - 1) \Delta_x}{2} + \frac{l_x}{2} \leq \frac{W_f}{2} - r_{tube} \tag{C10a: X 方向}$$

$$\frac{(n_y - 1) \Delta_y}{2} + \frac{w_y}{2} \leq \frac{D_f}{2} - r_{tube} \tag{C10b: Y 方向}$$

代入：X 跨 = 355 ≤ 570 ✓, Y 跨 = 285 ≤ 295 ✓（不含决策变量，始终满足）

### C11 — 拾取位在框架外部（v2.0 新增）

拾取点 $(x_c, y_c)$ 必须在框架 XY 矩形外至少 $m_{pick}$：

$$\max\!\left(x_c - \tfrac{W_f}{2} - m_{pick},\; \left(y_f - \tfrac{D_f}{2}\right) - m_{pick} - y_c,\; y_c - \left(y_f + \tfrac{D_f}{2}\right) - m_{pick}\right) \geq 0$$

即拾取点至少有一个方向距框架边界 $\geq m_{pick}$。

> **动机**：v1.0 中缺少此约束，优化器将传送带移入框架内部 ($y_c=477.5$) 以“作弊”缩短 Pick→Place 距离，但物理上不可行。

### C12 — 拾取位工具球-立柱碰撞间隙（v2.0 新增）

拾取位的工具球不得与框架立柱碰撞，与 C6 相同的安全距离要求：

$$r_{tool} + r_{col} + \epsilon_{col} - \|\mathbf{t}_{pick}^{xy} - \text{col}_k^{xy}\| \leq 0, \quad \forall k \tag{C12}$$

其中 $\mathbf{t}_{pick} = (x_c,\; y_c,\; z_{conv} + h_z/2 - \delta_{tool})$。

> **动机**：v1.0 + C11 修复后的最优解 ($x_c=580, y_c=255$) 拾取工具球距柱 2 仅 102mm，远小于所需 325mm。C12 纠正了此问题。

## 9 求解方法

### 9.1 算法选择

使用 **MATLAB `fmincon`**（SQP 算法），配合**多起始点**策略避免局部最优：

1. 以当前布局 $\mathbf{x}_0 = (700, 750, -800, 200)^T$ 作为首个起始点；  
2. 在边界框内均匀采样 $N_{start} - 1$ 个随机起始点；    
3. 对每个起始点运行 SQP 求解，保留可行解中目标值最小者。

若无优化工具箱，回退为 **罚函数** + `fminsearch`（Nelder-Mead 单纯形法）。

### 9.2 求解规模

| 指标 | 值 |
|------|----|
| 决策变量数 | 4 |
| 不等式约束（上界） | 114 |
| 等式约束 | 0 |
| 多起始点数 | 100 |
| 目标函数梯度 | 有限差分（步长 0.5mm） |

### 9.3 后处理

1. **可行性报告**：逐条检查所有约束的裕量；  
2. **灵敏度分析**：计算目标函数在最优解处对各变量的偏导数 $\partial f / \partial x_i$；  
3. **可视化**：俯视图和侧视图对比当前布局与最优布局；  
4. **参数映射**：将最优 $(y_f, x_c, y_c, h_p)$ 反算为 v15 代码中的配置参数。
